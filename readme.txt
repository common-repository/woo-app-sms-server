APPSMSServer - Your free personal pocket SMS Server uses an API interface and an android device to act as your sms gateway at lowests costs. It can communicate to your WooCommerce to send sms to your clients.
 
== Description ==
 
APPSMSServer - Your free personal pocket SMS Server uses an API interface and an android device to act as your sms gateway at lowests costs. It can communicate to your WooCommerce to send sms to your clients.

== Installation ==
 
Install the appsmsserver module on your website, activate the module, navigate in wp admin nav menu to APPSMSServer/Settings to configure it.

== Frequently Asked Questions ==
